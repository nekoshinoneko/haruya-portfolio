サイト管理

プロフィールサイトの内容（content.json）を編集して GitHub に保存するページです。保存するまで公開側は変わりません。トークンはこのブラウザの localStorage にのみ保存されます。

接続設定（GitHub）

GitHub → Settings → Developer settings → Fine-grained personal access tokens で、このリポジトリだけに Contents(Read and write) を許可したトークンを作成してください。

GitHub上の admin.html を書き換えてパスワードを変更します（接続設定が必要）。反映まで1〜2分かかります。

公開状態

ワンクリックで公開サイトを「メンテナンス中」表示に切り替えます。このボタンだけは押すとすぐにGitHubへ保存・反映されます。
